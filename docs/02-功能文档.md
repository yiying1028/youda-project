## 2. 学习社区模块

### 2.1 帖子发布

**功能描述：** 用户发布学习相关的帖子

**操作流程：**
1. 用户点击"发布帖子"
2. 填写标题、选择分类、编辑内容
3. 可上传图片
4. 点击发布

**字段说明：**

| 字段 | 必填 | 说明 |
|------|------|------|
| 标题 | 是 | 5-100字 |
| 分类 | 是 | 选择学科分类 |
| 内容 | 是 | 支持文字和图片 |

### 2.2 帖子列表

**功能描述：** 浏览帖子列表

**功能点：**
- 显示帖子标题、作者、时间、浏览数
- 按分类筛选
- 按最新/最热排序
- 分页加载

### 2.3 帖子详情

**功能描述：** 查看帖子完整内容

**展示内容：**
- 标题、内容、图片
- 作者信息
- 发布时间
- 点赞数、收藏数
- 评论列表

### 2.4 评论功能

**功能描述：** 对帖子发表评论

**功能点：**
- 发表评论
- 删除自己的评论
- 评论列表展示

### 2.5 点赞收藏

**功能描述：** 点赞和收藏帖子

**功能点：**
- 点赞/取消点赞
- 收藏/取消收藏
- 在个人中心查看收藏列表

## 3. 学习资料库模块

### 3.1 资料列表

**功能描述：** 浏览学习资料

**展示内容：**
- 资料名称
- 文件类型、大小
- 学科、年级分类
- 下载次数

**筛选条件：**
- 按学科筛选
- 按年级筛选
- 关键词搜索

### 3.2 资料详情

**功能描述：** 查看资料详细信息

**展示内容：**
- 资料名称、描述
- 文件信息
- 上传时间
- 下载按钮

### 3.3 资料下载

**功能描述：** 下载学习资料

**操作流程：**
1. 用户点击下载按钮
2. 系统返回文件
3. 下载次数+1

**支持格式：** PDF、Word、Excel、PPT、图片、压缩包

## 6. 错题本模块

### 6.1 错题收录

**功能描述：** 用户将做错的题目添加到错题本

**操作流程：**
1. 用户点击"添加错题"
2. 选择学科、年级
3. 填写题目内容（支持文字和图片）
4. 填写错误原因和正确答案
5. 点击保存

**字段说明：**

| 字段 | 必填 | 说明 |
|------|------|------|
| 学科 | 是 | 选择学科分类 |
| 年级 | 是 | 选择年级 |
| 题目内容 | 是 | 支持文字和图片 |
| 我的错答 | 否 | 记录自己的错误答案 |
| 正确答案 | 是 | 填写正确答案和解析 |
| 错误原因 | 否 | 如粗心、知识点不熟等 |

### 6.2 错题列表

**功能描述：** 浏览和管理错题

**功能点：**
- 按学科、年级筛选
- 按添加时间排序
- 关键词搜索
- 删除错题
- 编辑错题

### 6.3 错题复习

**功能描述：** 对错题进行复习巩固

**功能点：**
- 随机抽取错题进行复习
- 可按学科筛选复习范围
- 标记"已掌握"或"仍不会"
- 已掌握的题目不再出现在复习中

## 7. 每日打卡与积分模块

### 7.1 每日签到

**功能描述：** 用户每日签到获取积分

**操作流程：**
1. 用户点击"签到"按钮
2. 系统记录签到并发放积分
3. 显示连续签到天数

**积分规则：**
- 每日签到获得5积分
- 连续签到7天额外奖励20积分

### 7.2 积分获取与消费

**功能描述：** 通过学习行为获取积分，积分可用于平台消费

**积分获取方式：**

| 行为 | 积分 | 说明 |
|------|------|------|
| 每日签到 | +5 | 每天一次 |
| 发布帖子 | +3 | 每天上限3次 |
| 发表评论 | +1 | 每天上限5次 |
| 完成课程章节 | +10 | 完成一个章节 |
| 添加错题 | +2 | 每天上限5次 |

**积分消费方式：**
- 下载付费资料扣除对应积分

### 7.3 积分排行榜

**功能描述：** 展示积分排名，激励学习

**功能点：**
- 显示积分排名前20的用户
- 展示用户昵称、头像、积分数
- 显示自己的当前排名

## 8. 用户模块

### 8.1 用户注册

**功能描述：** 新用户注册

**操作流程：**
1. 输入用户名（4-20字符）
2. 输入密码（6-20字符）
3. 确认密码
4. 点击注册

### 8.2 用户登录

**操作流程：**
1. 输入用户名
2. 输入密码
3. 点击登录
4. 登录成功跳转首页

### 8.3 个人中心

**功能描述：** 管理个人信息

**功能点：**
- 修改头像
- 修改昵称
- 修改密码
- 查看我的帖子
- 查看我的收藏
- 查看我的错题本
- 查看学习记录
- 查看积分与签到
