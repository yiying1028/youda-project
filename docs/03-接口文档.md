# 优答·中小学答疑网 - 接口文档

## 1. 接口规范

### 1.1 基本信息

- **基础URL：** `http://localhost:8080/api`
- **数据格式：** JSON
- **字符编码：** UTF-8

### 1.2 请求头

| 参数名 | 必填 | 说明 |
|--------|------|------|
| Content-Type | 是 | application/json |
| Authorization | 否 | Bearer {token}，登录接口必填 |

### 1.3 统一响应格式

```json
{
    "code": 200,
    "message": "success",
    "data": {}
}
```

### 1.4 响应状态码

| 状态码 | 说明 |
|--------|------|
| 200 | 成功 |
| 400 | 参数错误 |
| 401 | 未登录 |
| 403 | 无权限 |
| 404 | 资源不存在 |
| 500 | 服务器错误 |

### 1.5 分页参数

| 参数 | 说明 |
|------|------|
| current | 当前页码，默认1 |
| size | 每页数量，默认10 |

### 1.6 分页响应

```json
{
    "code": 200,
    "message": "success",
    "data": {
        "records": [],
        "total": 100,
        "current": 1,
        "size": 10
    }
}
```
